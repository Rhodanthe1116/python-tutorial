## **while與for比較表**

| **名稱**   | **while**                           | **for**             |
| -------- | ----------------------------------- | ------------------- |
| **計數變數** | **需另建立計數器**                         | **自帶計數器，不受迴圈內任何影響** |
| **省略**   | **無省略，皆須寫清楚**                       | **可將初始值與增加值省略**     |
| **特點**   | **在條件判斷的位置，只要不為false、0、null皆會進入迴圈** | **只要有範圍就可以使用**      |
| **使用時機** | **在不需要額外變數時進入迴圈**                   | **利用變數對範圍內的資料進行處理** |
