# 更寬廣的範圍

for**不一定要用 range( )**

```python
for color in 'red','green','blue':
        print(color)
```

```python
for num in 'red',1,'green',2,'blue':
        print(num)
```

```python
x = 'variable'
for var in x,'red',1,'green',2,'blue':
        print(var)
```

```python
for char in 'hello':
        print(char)
```
