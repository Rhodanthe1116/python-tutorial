# 二維串列

## 暖身時間

學期末到了，老師要計算每個學生的學期總成績， 該班級有三次考試，共有五位學生，每一列表示一 次考試，五位學生的成績，請算出每位學生的平均成績

```bash
# 範例輸入
17 36 48 87 63
92 73 64 33 81
88 76 32 55 47

# 範例輸出
65.66666666666667
61.666666666666664
48.0
58.333333333333336
63.666666666666664
```

## 二維串列

```python
data = [[17, 36, 48, 87, 63], [92, 73, 64, 33, 81], [88, 76, 32, 55,
47]]
```

![](<../../.gitbook/assets/image (84).png>)

此時data就是一個二維串列！

另外一種寫法

```python
data = [
    [17, 36, 48, 87, 63],
    [92, 73, 64, 33, 81],
    [88, 76, 32, 55, 47]
]    
```

別忘了在串列中，元素與元素之間要用 , 隔開
