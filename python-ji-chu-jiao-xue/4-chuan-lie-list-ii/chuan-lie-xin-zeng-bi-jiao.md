## 比較一

```python
a = [1, 2, 3]
a.append(4)
print(a)
# 得到: [1, 2, 3, 4]
a += 4
# error!
print(a)
```

```python
a = [1, 2, 3]
a.append([4])
# 得到: [1, 2, 3, [4]]
a += [4]
# 得到: [1, 2, 3, [4], 4]
print(a)
```

#### \*串列中還可以插入串列!!

## 比較二

* a = \[1, 2, 3]
* b = \[4, 5, 6]

將 b 串列內的元素加入 a 串列

```python
a += b
# [1, 2, 3, 4, 5, 6]
for d in b:
    a.append(d)
# [1, 2, 3, 4, 5, 6]
```

```python
a += [b]
# [1, 2, 3, [4, 5, 6]]
a.append(b)
# [1, 2, 3, [4, 5, 6]]
```
