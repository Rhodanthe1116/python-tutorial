# 串列的中括號

在python的串列中，可以在中括號內將串列的元素逐一取 出並作運算，再以這些運算結果建立一個新的串列

```python
data = [int(d) for d in input().split()]
```

\=

```python
data = []
for d in input().split():
data.append(int(d))
```

## 練習時間

輸入一行有5個整數，每個整數間隔一個空格，請 將這5個數都+10後存成串列再輸出

```bash
# 範例輸入:
17 36 48 80 77

# 範例輸出:
[27, 46, 58, 90, 87]
```

## 中括號內也可以添加 if 進行篩選

```python
data = [int(d) for d in input().split() if int(d) > 10]
```

等同於

```python
data = []
for d in input().split():
    if int(d) > 10:
        data.append(int(d))
```
