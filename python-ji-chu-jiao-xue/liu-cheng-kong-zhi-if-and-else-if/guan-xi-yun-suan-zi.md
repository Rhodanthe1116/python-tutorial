# 關係運算子

**條件式的運算結果只會有兩種：**

* **真 ( True )**
* **假 ( False )**

## **範例**

```python
if True:
	print('True')
else:
	print('False')

if False:
	print('True')
else:
	print('False')
```

```bash
# 輸出

True
False
```

## **關係運算子有哪些？**

| **意義**   | **關係運算子** |
| -------- | --------- |
| **相等**   | **==**    |
| **不相等**  | **!=**    |
| **大於**   | **>**     |
| **小於**   | **<**     |
| **大於等於** | **>=**    |
| **小於等於** | **<=**    |

## **關係運算子是種二元運算子**

**二元運算子 : 運算時需要兩個運算元**

* **運算子 : operator**
* **運算元 : operand**
